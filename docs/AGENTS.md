# docs

## Purpose
Claude Flow Monitor의 기획, 디자인, 개발 가이드 문서를 포함하는 최상위 문서 디렉토리. 프로젝트의 모든 비코드 문서가 하위 디렉토리에 주제별로 분류되어 있다.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `plans/` | 현황 분석, PRD 등 기획 문서 (see `plans/AGENTS.md`) |
| `design/` | UI/UX 디자인 가이드, i18n 설계 (see `design/AGENTS.md`) |
| `dev-guide/` | 기술 아키텍처, 개발 가이드 (see `dev-guide/AGENTS.md`) |

## For AI Agents

### Working In This Directory
- 문서 버전 형식: `X.Y.Z-draft` (수정 시 minor 버전 올리고 `최종 수정일` 갱신)
- 모든 문서는 한국어로 작성
- ASCII 다이어그램 사용 (외부 이미지 파일 지양)
- 섹션 번호 일관성 유지 (새 섹션 추가 시 이후 번호 재조정)

### Cross-References
- PRD의 기능 번호(F1~F13)는 다른 문서에서 참조됨 — 변경 시 전체 검색 필요
- 디자인 가이드의 CSS 변수명(`--cfm-*`)은 아키텍처 문서와 동기화
- i18n 키 구조는 아키텍처 문서의 메시지 프로토콜과 연동
